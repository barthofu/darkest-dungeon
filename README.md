**Bienvenue dans le Donjon Obscur !** Votre objectif est simple : *échappez-vous* de ce donjon labyrinthique tout en étant plongé dans une intense *pénombre* ! En plus de cette obscurité, de nombreux *ennemis* rôdent dans les parages et des temples magiques conférant divers bonus et malus sont disséminés un peu partout !

Nécessaire :

- python 3 ou supérieur
- pip pour python 3 ou supérieur
- pygame version 1.9.6 ou sup rieur

## Installation :

- [Installez la dernière version de python 3 ici](https://www.python.org/downloads/)
- *pip3 devrait être installé par défaut avec python 3* 
- Installez la dernière version de pygame:
```bash
pip3 install pygame
```

## Exécution :

```bash
python3 /location/du/projet/darkest\_dungeon.py
```

## Informations complémentaires :

- Vous pouvez afficher le labyrinthe en entier sans la pénombre avec la touche clavier `L` (assez sensible) pour observer la génération du labyrinthe et le déplacement des ennemis.
- Bien que la touche pour tirer soit `A`, il se peut que vous deviez utiliser `Q`
- Dans  le  dossier  `prototypes`  se  trouve  le  script  `labyrinthe.py`  qui  contient  le  premier programme prototype de l’algorithme de génération et de résolution du labyrinthe. Le lancer génèrera directement le labyrinthe. Vous pouvez ensuite cliquer sur `Q` (ou  `A` ) pour lancer l’algorithme de résolution.
